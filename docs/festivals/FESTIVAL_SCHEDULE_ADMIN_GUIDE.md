# Festival Schedule Admin Guide

Open the Schedule tab from the festival admin workspace. Desktop shows a multi-stage timeline with visible day tabs and conflict markers. Mobile uses a day selector, stage selector and vertical agenda.

## Common tasks
1. Configure stage hours from Operations/Stages or schedule mutations.
2. Add a manual slot from **Manual slot**.
3. Preview a template from **Templates** before applying.
4. Select an item to open the inspector and edit title, type, times, notes or move it to unscheduled.
5. Review conflict warnings and readiness summaries.
6. Use **Publish** to review item counts, unscheduled counts and blockers before publishing.

Drafts are private. Public pages only consume the published projection.
